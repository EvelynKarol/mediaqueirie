Meu site

Sobre

Este é um site responsivo, cuja finalidade é apresentar a adaptação do layout para diferentes dispositivos, garantir acessibilidade e boa experiência de uso.

O que utilizamos

Neste contexto utlizamos o Media Queries, cuja definação é regras condicionais em CSS para aplicar estilos conforme características do dispositivo. Além das estruturas básicas do HTML e CSS.
